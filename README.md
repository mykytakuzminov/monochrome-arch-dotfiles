## üíª Installation

### 1. Clone the repository

```bash
git clone https://github.com/mykytakuzminov/arch-monochrome-dotfiles.git
cd arch-monochrome-dotfiles
```

### 2. Make the installer executable

```bash
chmod +x install.sh
```

### 3. Run the installer

```bash
./install.sh
```

### 4. Reboot the system
```bash
reboot
```
